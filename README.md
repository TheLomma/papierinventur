# Papierrollen Inventur

Mobile Web-App für die Papierrollen-Inventur der Rheinische Post Mediengruppe.

## Deployment

1. Repository zu GitHub pushen
2. Bei [vercel.com](https://vercel.com) mit GitHub verbinden
3. Repository auswählen und "Deploy" klicken

## Als App installieren

- **iPhone**: Safari → Teilen → "Zum Home-Bildschirm"
- **Android**: Chrome → Menü → "Zum Startbildschirm hinzufügen"
